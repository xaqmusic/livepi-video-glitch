# Clips

Full-resolution gig footage goes here (gitignored -- see `.gitignore` and
`docs/architecture.md`'s "Asset strategy"). This directory is checked into
git only as an empty placeholder (`.gitkeep`) plus this README.

For now, get real footage onto a machine by copying it directly (USB stick,
`scp`, or `rsync` straight from wherever it was filmed/edited) -- there's no
asset-fetch tooling yet since no real footage library exists. Revisit if that
becomes painful (see the "longer-term" note in `docs/architecture.md`).

See `samples/` for the small clips checked into git for development.
